Checks if there are at least two trees compatible with the given constraints, while keeping the root fixed.

Parameters
num_leavesThe number of leaves of the tree
constraintsThe constraints on the leaves. The constraints must only contain valid leaf indices.
root_leafThe fixed 'root leaf' which will be placed to the right of the root to avoid isomorphic, differently rooted trees.
Returns
true iff there are at least two trees compatible with the constraints.
